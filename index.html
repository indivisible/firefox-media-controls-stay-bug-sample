Firefox bug from circa version 95: external media controls (MPRIS controls, windows media control overlay) are sometimes not removed when the media is removed from the DOM.

While simple test cases seem to work fine, youtube embeds will trigger the bug. Even reloading the offending tab is not enough to remove the stuck controls: only closing the tab or unloading it with an extension seems to fix it.
